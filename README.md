## 🧑‍💻 Local Setup

Clone the repository:

```bash
git clone https://github.com/adnanpal/Portfolio.git
```

Navigate to the project directory:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
